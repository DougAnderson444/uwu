# Build from source

## For CLI:

Navigate to ./crates/uwu-cli and run

```
cargo build --release
```

## For WASM web & node:

Navigate to ./crates/uwu-wasm/README.md and follow the directions there.

## Final Build Step

Then run `npm run build`
